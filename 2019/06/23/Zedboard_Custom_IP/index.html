Zedboard_Custom_IP

Posted on 2019-06-23

ZedBoard turn on Light by launching SDK and Custom IP


1、Create Project

Project Name -> RTL Project -> Next -> Choose “Boards / ZedBoard Zynq Evaluation and Development Kit” -> Finish


2、Custom IP

Create Block Design -> add IP -> search “ZYNQ7 Processing System -> Run Block Automation -> OK

Double Click ZYNQ -> Add UART 0 port

Create and package New IP

Edit in IP Package ： Right click “Blinker_ip_0” and choose Edit in IP Package -> ok

Edit blink_ip_v1_0.v -> Edit blink_ip_v1_0_S00_AXI.v -> Merge changes

Edit blink_ip_v1_0.v ： edit line 17, 18, 71, 73, 74

Edit blink_ip_v1_0_S00_AXI.v ： edit line 17, 18, 376, 378, 403

Run Synthesis -> close the Project -> then you can add your custom IP

Refresh IP Catalog -> delete old blinker_ip_0 and add again -> Run Connection Automation

Create Ports input [7:0] switches and ouput [7:0] leds -> connect with module -> Create HDL Wrapper -> OK

Run Synthesis -> Run Implementation -> Open Implementation Design -> Set Ports -> Ctrl + S -> constraints.xdc

Run Implementation -> Generate Bitstream -> Open Hardware Manager -> Auto Connect -> Program


3、Launch SDK

Project Export hardward (check include bitstream) -> launch SDK -> New C project

Now your leds will be turned on by switches.
